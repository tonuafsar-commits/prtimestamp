# PRTimestamp

A static browser tool for converting Premiere Pro marker CSV exports into clean timestamp text for YouTube chapters, notes, captions, and edit reviews.

## GitHub Pages

This site is ready to publish from the repository root. The homepage is `index.html`.

In GitHub, open the repository settings, choose **Pages**, then set:

- Source: **Deploy from a branch**
- Branch: **main**
- Folder: **/(root)**
